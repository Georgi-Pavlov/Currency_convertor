# Currency Converter with GUI
A simple Python project that converts currency amounts using live exchange rates from an external API and provides a graphical user interface built with Tkinter.
This project started as a console application and evolved into a small desktop app as a way to practice:

- Working with external APIs
- Structuring code into modules
- Handling errors properly
- Building a basic GUI with Tkinter

Features

- Fetches real-time exchange rates from an online API
- Converts between two currencies
- Input validation and error handling
- Simple and clean Tkinter interface

Possible Improvements

- Dropdown list with available currencies
- Better UI styling
- Packaging the app as an executable

API Used

Exchange rates are provided by: https://api.exchangerate-api.com
